## How to Use

### Install dependencies

```bash
npm install
```

### Seed DB with random data

```bash
npm run seed
```

### Run the development server

```bash
npm run dev
```

### Add mongo uri

Create a .env file with your mongo db uri.<br />
In the project, you can find a sample.env with an example
